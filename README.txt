Movie trailer website 05/14/2017

Introduction
============================================
This project creates a website to play movie trailers, it 
contains:

- media.py
	class for organizing movie data
- entertainment_center.py
	list of movies and information, run this module to 
	create website.
- fresh_tomatoes.py
	website formating

Steps
============================================
1. Download the zip file and unzip
2. You will have to install Python 2.7.12 if you don’t have one
3. run ‘python entertainment_center.py’ in command line
4. The browser should open with movies page
